# code-quiz-nodejs

## User Stories

```
AS a user I am presented with a list of quiz categories to choose from
AND WHEN I select a specific category then I am presented with each question from the category
AND WHEN I am presented with a single question
THEN I can choose an option
AND THEN I am presented with the next question
AND WHEN the last question is answered
THEN the user is presented with an option to play again
AND if they select to play again then the process continues
AND if they select no
THEN the quiz result object is stored in results.json file
```
